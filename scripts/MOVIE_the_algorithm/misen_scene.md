# Mise-en-scène: THE ALGORITHM

## 1. 전반적 시각 기조 (Visual Tone)
- **High-Contrast Noir**: 극단적인 명암 대비. 데이터의 흐름을 나타내는 푸른색 발광체와 절대적 어둠의 충돌.
- **Glitch Aesthetic**: 알고리즘이 현실을 잠식할 때 나타나는 미세한 화면 떨림, 색차 왜곡(Chromatic Aberration)의 실재화.

## 2. 주요 장소 설계 (Key Locations)

### A. Global Data Corp (심장부)
- **질감**: 차가운 스테인리스 스틸, 초정밀 유리.
- **조명**: 바닥에서 올라오는 무감각한 백색광. 그림자가 거의 없는 결벽증에 가까운 공간.
- **분위기**: 기계의 위엄과 인간의 왜소함을 극대화.

### B. Sophie's Container (은신처)
- **질감**: 녹슨 철, 매달린 전선들, 습기 찬 공기.
- **조명**: 수십 개의 모니터에서 뿜어 나오는 다채롭고 무질서한 안광.
- **분위기**: 현실로부터 버려진 자의 마지막 요새. '아날로그의 습격'을 준비하는 공간.

### C. The Glitched Stock Exchange (거래소)
- **질감**: 균열이 간 대리석 바닥, 공중에 흩날리는 데이터 리본.
- **조명**: 경고등(붉은색)과 알고리즘(푸른색)의 혼란스러운 깜빡임.
- **분위기**: 자본주의의 종말을 시각화한 지옥의 풍경.
