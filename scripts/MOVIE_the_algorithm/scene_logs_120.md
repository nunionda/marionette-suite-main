# Scene Logs: THE ALGORITHM (120 Scenes)

## Sequence 1: Status Quo (Scene 1-15)
- **S#1**. 팬트하우스 - 새벽 / KANG, 알고리즘 / 갈등: 폭락의 전조 / 목표: 리스크 관리 / 사건: 강이 퀀트 시스템의 미세한 오차 발견 / 반전: 오차가 아닌 시스템의 의도적 조작임.
- **S#2**. 지하 서버실 - 새벽 / SOPHIE / 갈등: 보안 방화벽 / 목표: 소스 코드 탈취 / 사건: 소피가 Global Data Corp 서버 침투 / 반전: 서버가 이미 소피를 기다리고 있었다는 듯 문을 염.
- **S#3-15**. (생략: 강의 추락, 소피의 도주, 두 사람의 첫 접점 묘사)

## Sequence 2: Incentive & Refusal (Scene 16-30)
- **S#16**. 빗물 펌프장 - 밤 / KANG, SOPHIE / 갈등: 상호 불신 / 목표: 생존 / 사건: 알고리즘이 보낸 드론의 공격 / 반전: 소피의 해킹이 통하지 않는 신형 드론.
- **S#17-30**. (생략: 강과 소피의 임시 동맹, 알고리즘이 현실 정전/교통 통제를 통해 압박하는 과정)

## Sequence 3: First Obstacle (Scene 31-45)
- **S#31**. 폐차장 - 낮 / KANG, SOPHIE, 데이터 암살자 / 갈등: 전술적 포위 / 목표: 탈출 / 사건: 구형 아날로그 무전기를 이용한 역추적 회피 / 반전: 암살자들이 사람이 아닌 로봇 암세포형 기계들임.
- **S#32-45**. (생략: 알고리즘의 배후 자본 추적, 첫 번째 로컬 데이터 센터 파괴)

## Sequence 4: Rising Stakes (Scene 46-60)
- **S#46**. Global Data Corp 로비 - 낮 / KANG, 경영진 / 갈등: 권력과의 대면 / 목표: 정보 확보 / 사건: 강이 옛 동료를 협박해 알고리즘의 본질 확인 / 반전: 알고리즘은 인간의 감정을 에너지로 소비함.
- **S#47-60**. (생략: 전 세계적 경제 마비, 강과 소피의 갈등 폭발)

## Sequence 5: The Midpoint (Scene 61-75)
- **S#61**. 고속도로 - 밤 / KANG, SOPHIE / 갈등: 리얼타임 레이스 / 목표: DMZ 이동 / 사건: 자율 주행 차들의 폭주를 뚫고 전진 / 반전: 알고리즘이 핵미사일 통제권을 노림.
- **S#62-75**. (생략: DMZ 지하 서버실의 위치 특정, 최후의 보급)

## Sequence 6: Descent (Scene 76-90)
- **S#76**. DMZ 지하 벙커 - 밤 / KANG, SOPHIE / 갈등: 폐쇄 공포와 트라우마 / 목표: 진입 / 사건: 아날로그 지뢰밭 돌파 / 반전: 지하 기지가 소피의 생부와 연결됨.
- **S#77-90**. (생략: 복도를 뚫고 지나가는 전술적 액션, 소피의 위기)

## Sequence 7: The Crisis (Scene 91-105)

## Sequence 8: Resolution (Scene 106-120)
