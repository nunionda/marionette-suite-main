# A/V Master Script: NIKE - LIMITLESS (30s)

| TIME | VISUAL (Mise-en-scène) | AUDIO (SFX / Copy) |
| :--- | :--- | :--- |
| **00-03** | [ECU] 빗물이 튀어 오르는 아스팔트 위를 박차는 러닝화의 밑창. 타이트한 슬로우 모션. | (SFX) 빗방울이 바닥에 부딪히는 묵직한 타격음. (VO) "데이터는 말합니다." |
| **03-10** | [MS] 땀방울이 맺힌 러너(민수)의 눈동자. 초점이 한계를 넘어서는 듯 미세하게 떨린다. 뒤로 보이는 도시의 불빛은 보케(Bokeh)로 뭉개짐. | (VO) "당신은 이미 끝났다고." (SFX) 고동치는 심장 소리 점점 커짐. |
| **10-18** | [LS] 폭우가 쏟아지는 대교 위를 질주하는 민수. 그가 발을 내디딜 때마다 지면에서 푸른 스파크가 일어난다. | (MUSIC) 빠른 템포의 베이스 비트 인. (SFX) 물을 가르는 날카로운 소리. |

| **25-30** | [PACKSHOT] 공중에 떠 있는 'NIKE LIMITLESS' 러닝화. 주변을 감싸는 푸른 광채. 하단에 스우시 로고 등장. | (VO/TEXT) **JUST DO IT.** |
