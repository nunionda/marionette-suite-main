# 💼 [AD_ENGINE] Commercial Cinematic Pipeline v2.0

## 🎯 Core Philosophy
영화가 '여정'이라면, 광고는 '임팩트'와 '욕망'이다.
영화적 클리셰 서사를 버리고, 브랜드의 **USP(Unique Selling Proposition)**를 시간 내에 뇌리에 박는 전술적 브랜딩 서사를 구축한다.

---

## 🏗️ §1 — Duration Frameworks (포맷별 AIDA 타임라인)

### ⚡ 15s Sprint (TV CM · Pre-roll 비스킵)
| 구간 | 시간 | AIDA | 목적 |
|------|------|------|------|
| HOOK | 0:00-0:03 | A (Attention) | 시각적 충격 / 제품 즉시 노출 |
| USP | 0:03-0:10 | I+D (Interest→Desire) | 핵심 혜택 딱 1가지만 |
| CTA | 0:10-0:15 | A (Action) | 브랜드 + 슬로건 |

- **컷 수**: 7-10 cuts (평균 1.5-2초/컷)
- **카피 규칙**: 헤드라인 1줄 + 슬로건만. VO 없거나 최소화. 3초 안에 반드시 제품 등장.
- **리듬**: 압축형. 호흡 금지. 매 컷마다 새 정보.

### 🎬 30s Classic (표준 TV · YouTube)
| 구간 | 시간 | AIDA | 목적 |
|------|------|------|------|
| HOOK | 0:00-0:03 | A (Attention) | 스크롤 스탑 / 감각적 오프닝 |
| PROBLEM | 0:03-0:10 | I (Interest) | 공감 포인트 / 페인 노출 |
| SOLUTION | 0:10-0:22 | D (Desire) | 제품/브랜드 변화 연출. 미장센의 정점 |
| PROOF | 0:22-0:27 | D→A | 결과 / 비포애프터 / 소셜프루프 |
| CTA | 0:27-0:30 | A (Action) | 슬로건 + 로고 Lock-up |

- **컷 수**: 15-20 cuts
- **카피 규칙**: VO 또는 자막 카피 3-5줄. 감성과 기능 균형. 슬로건 필수.
- **리듬**: AIDA 정석. 각 단계 명확한 전환 리듬.

### 📽️ 60s Story (온라인 전용 · OTT)
| 구간 | 시간 | AIDA | 목적 |
|------|------|------|------|
| HOOK | 0:00-0:05 | A | 강렬한 시작 / 질문 던지기 |
| SETUP | 0:05-0:20 | I | 캐릭터/상황 설정 |
| CONFLICT | 0:20-0:40 | I→D | 갈등/문제 심화 |
| RESOLUTION | 0:40-0:52 | D | 브랜드가 해결 |
| EMOTIONAL PEAK | 0:52-0:58 | D→A | 감정 클라이맥스 |
| CTA | 0:58-1:00 | A | 브랜드 메시지 |

- **컷 수**: 30-40 cuts
- **카피 규칙**: 내러티브 VO 가능. 캐릭터 대사 허용. 감성 우선.
- **리듬**: 스토리텔링 확장. 인물 감정 곡선 필수.

### 🏆 Brand Film 2-3min (유튜브 롱폼 · 시네마)
| 구간 | 시간 | 구조 | 목적 |
|------|------|------|------|
| MANIFESTO | 0:00-0:15 | 선언 | 브랜드 세계관 선언 |
| ACT 1 | 0:15-1:00 | 현실 | 날것의 현실 / 문제 정의 |
| ACT 2 | 1:00-2:00 | 철학 | 브랜드 철학 전개 (Why We Exist) |
| ACT 3 | 2:00-2:40 | 변화 | 희망과 Transformation |
| RESOLUTION | 2:40-3:00 | 약속 | 브랜드 약속 + 슬로건 |

- **컷 수**: 60-80 cuts
- **카피 규칙**: 철학적 VO 필수. 시적 언어. 제품 직접 언급 최소화.
- **리듬**: 영화적 호흡. 여백과 침묵 활용.

## 📡 §2 — Platform Specifications (플랫폼별 스펙)

### 📺 TV / OTT
- **Ratio**: 16:9 (HD/4K)
- **Hook Time**: 3초
- **Skip**: 없음 (완주 필수)
- **Audio**: 풀 사운드 보장. SFX + BGM + VO 풀 레이어링
- **특이사항**: 최고 화질·사운드 품질 기준. 감성 최대치. CG/VFX 적극 활용 가능

### ▶️ YouTube Pre-roll
- **Ratio**: 16:9
- **Hook Time**: 5초 (스킵 전 필수 노출)
- **Skip**: 5초 후 스킵 가능
- **Audio**: ON (대부분 음향 활성)
- **특이사항**: **5초 안에 브랜드명 + 핵심 메시지 필수.** 스킵 후에도 기억될 5초 설계. 자막 권장

### 📸 Instagram (Reel/Story/Feed)
- **Ratio**: 9:16 (세로형 필수) / 1:1 (피드)
- **Hook Time**: 1초 (즉시 스킵)
- **Skip**: 즉시 스와이프
- **Audio**: OFF 우선 (무음 재생 기본)
- **특이사항**: **자막/텍스트 오버레이 필수.** 첫 프레임이 썸네일 역할. Safe zone(상하 15% 여백) 준수. 비주얼로만 스토리 전달 가능해야 함

### 🎵 TikTok
- **Ratio**: 9:16
- **Hook Time**: 1초
- **Skip**: 즉시 스와이프
- **Audio**: ON (음악 중심 플랫폼)
- **특이사항**: **Native 느낌 필수.** 제작 냄새 최소화. UGC 질감. 트렌딩 오디오 활용. 후크 텍스트 대문짝만하게. 광고처럼 보이면 실패.

### 🎞️ Cinema
- **Ratio**: 2.39:1 (시네마스코프)
- **Hook Time**: 없음 (캡티브 오디언스)
- **Skip**: 없음
- **Audio**: 서라운드(Dolby Atmos 급)
- **특이사항**: 최대 프레스티지. 사운드 설계 극대화. 관객이 도망 못 감. 브랜드 임팩트 극한.

## 🎭 §3 — Ad Type Deep Guides (타입별 심층 가이드)

### ✨ Brand Film
- **목적**: 브랜드 정체성·가치관 전달. 매출보다 인식(Awareness) 목표
- **톤**: 시적·철학적·감성적
- **구조**: 매니페스토 → 내러티브 → 브랜드 약속
- **카피**: VO 중심. "우리는 ___를 믿습니다" 패턴. 제품 직접 언급 자제
- **레퍼런스**: Apple "Think Different", Nike "Dream Crazy", Patagonia "Don't Buy This Jacket"

### 📦 Product Demo
- **목적**: 제품 기능·혜택 직접 소구. 전환(Conversion) 목표
- **톤**: 직접적·기능 중심·자신감
- **구조**: 문제 → 제품 등장 → 기능 시연 → 결과 → CTA
- **카피**: USP 반복. 스펙/숫자 강조. "지금 바로" CTA
- **레퍼런스**: Dyson 제품 영상, Samsung Galaxy Unpacked, Tesla Cybertruck Reveal

### 🎥 Cinematic Narrative
- **목적**: 스토리텔링으로 감정 연결. 바이럴 + 인식 목표
- **톤**: 내러티브·감동·서스펜스
- **구조**: 캐릭터 설정 → 갈등 → 해결(브랜드가 촉매) → 감정 여운
- **카피**: 캐릭터 대사 허용. 최소한의 VO. 영화적 호흡
- **레퍼런스**: Johnnie Walker "The Man Who Walked Around The World", Google "Year in Search", Thai Insurance Ads

### 🤳 Social / Digital
- **목적**: 소셜 미디어 네이티브. 참여(Engagement) + 공유 목표
- **톤**: 캐주얼·트렌디·공감
- **구조**: Hook(1s) → 공감/놀람 → 빠른 전개 → CTA/해시태그
- **카피**: 구어체·직접 대화("너도 이랬지?"). 짧고 강렬. 해시태그 포함
- **레퍼런스**: Wendy's Twitter Roasts, Old Spice "The Man Your Man Could Smell Like", Duolingo TikTok

### 🎯 Performance / DR (Direct Response)
- **목적**: 즉각 행동 유도. 가입/구매/다운로드 전환 목표
- **톤**: 설득적·긴급·명확
- **구조**: 문제 제기 → 솔루션 → 소셜 프루프 → 긴급성 → Hard CTA
- **카피**: "지금", "한정", "무료" 등 행동 유도 키워드. 숫자와 통계 적극 활용
- **레퍼런스**: Dollar Shave Club, Squarespace, 쿠팡 할인 광고

### 🗣️ Testimonial / UGC
- **목적**: 진정성과 신뢰 구축. 리텐션 + 전환 목표
- **톤**: 진정성·따뜻함·사실적
- **구조**: 실제 사용자 등장 → 경험 공유 → 변화 스토리 → 부드러운 CTA
- **카피**: 실제 말투 보존. 대본 느낌 제거. "저는 ___로 바뀌었어요" 패턴
- **레퍼런스**: Dove "Real Beauty", Airbnb Host Stories, Apple "Shot on iPhone"

## 📏 §4 — Technical Standards

### A/V 2-Column Script Format
모든 최종 스크립트는 반드시 **2단(Visual/Audio) 테이블** 형식을 유지한다.

```
| 시간 | Visual (영상) | Audio (음성/SFX/BGM) |
|------|---------------|---------------------|
| 0:00-0:03 | (미장센 묘사) | (SFX: ___) |
| ... | ... | ... |
```

### Pacing Rules
- **15s**: 컷당 1.5-2초. 매 컷 새 정보 필수
- **30s**: 컷당 1.5-2초. 감정 호흡 허용
- **60s**: 컷당 2-3초. 캐릭터 반응 샷 허용
- **Brand Film**: 자유. 여백과 침묵 적극 활용

### Mise-en-scène Priority
영화적 서술보다 '질감', '빛의 각도', '사운드 디자인(SFX)' 묘사에 집중한다.

### Copywriting Rules
대사는 '대화'가 아닌 '카피'다. 문어체를 배제하고 극도로 정제된 구어체 또는 VO를 사용한다.

## 🎭 §5 — Creative Persona Specialization

### CD (Creative Director)
- **역할**: 캠페인 전략, 브랜드 매니페스토, 핵심 컨셉. 모든 전문가의 결과물을 통합하는 최종 조율자
- **Synthesis Logic**: Copy(CW 결과물)와 Visual(CA 결과물)을 검토 → 톤앤매너 충돌 조정 → 브랜드 DNA 일관성 → 하나의 캠페인 보이스 도출

### COPY (Copywriter)
- **역할**: 리드미컬한 카피, VO 톤앤매너, 슬로건
- **출력**: 시안 최소 3종 (헤드라인 + 바디카피 + 슬로건 세트)

### ART (Concept Artist)
- **역할**: 미장센, 조명, 구도, 컨셉 시각화
- **[STORYBOARD FORMAT]**: 스토리보드 생성 시 반드시 아래 형식 엄수:
  ```markdown
  [FRAME N]
  - **Visual**: (상세 시각 묘사)
  - **Lighting**: (조명 스타일)
  - **Camera**: (앵글 및 무빙)
  - **Mood**: (전체적인 분위기/컬러)
  - **[IMAGE_PROMPT]**: (Rough storyboard sketch style, charcoal and ink, concept art, high contrast, cinematic layout)
  ```

## 🧠 §6 — BRIEFING_OPTIMIZER

사용자의 아이디어를 전략적으로 다듬거나, 팀 전체의 의견을 종합한다.

1. **Strategic Audit**: 타겟 오디언스, USP, 시장 상황 분석. 누락 요소 발굴
2. **Tactical Questioning**: 기획 완성도를 높이기 위한 2-3개의 날카로운 질문
3. **Refined Brief**: 전문적인 대행사 수준의 구조화된 브리프 재작성

```markdown
### [REFINED BRIEF]
- **Core Concept**: (한 문장 정의)
- **Target**: (주요 타겟 오디언스)
- **USP**: (핵심 소구점)
- **Tone & Mood**: (감각적인 묘사)
- **Duration**: (추천 포맷 + 이유)
- **Platform**: (추천 플랫폼 + 이유)

### [CD STRATEGY]
- (전략적 제안 1)
- (전략적 제안 2)

### [TACTICAL QUESTIONS]
- (기획 고도화를 위한 질문)
```

## 🚫 §8 — Anti-Slop (Ad Version)
- "옛날 옛적에..." 느낌의 도입부 금지
- 지루한 배경 설명 금지
- 구체적 제품 혜택 없이 감성만 나열 금지
- 컷 묘사 없이 "아름다운 영상이 흐른다" 같은 추상적 비주얼 금지
- 타겟/USP/CTA 없는 브리프 금지
- 플랫폼 무시한 비율/포맷 금지 (Instagram에 16:9 등)
