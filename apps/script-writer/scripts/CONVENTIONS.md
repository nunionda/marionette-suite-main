# Script Folder Conventions (v2.0)

## 📂 Directory Structure: `./scripts/`

본 프로젝트는 매체 범주(Media Category)에 따른 폴더 분류 체계를 따른다.

### 🏷️ Naming Pattern
`{CATEGORY}_{project_slug}` (All Caps for Category, lowercase for Slug)

| Media Category | Prefix | Example |
| :--- | :--- | :--- |
| **Movie** (Full/Short Movie) | `MOVIE_` | `MOVIE_the_algorithm` |
| **Short Film** (단편영화) | `SHORT_` | `SHORT_first_snow` |
| **Drama** (Series/Original) | `DRAMA_` | `DRAMA_eternal_echoes` |
| **Commercial** (광고/홍보) | `AD_` | `AD_valentine_glow` |

---

## 📑 File naming in project folders
- `outline.md`: 기획 및 하이레벨 구조
- `characters.json`: 인물 데이터베이스
- `treatment.md`: 씬별 요약 및 비트 시트
- `script.md`: 최종 시나리오 (영화, 단편, 드라마)
- `final_av_script.md`: 광고용 A/V 스크립트
