# Pilot Script: ETERNAL ECHOES (영원의 메아리)

**S# 1. DATA CENTER - NIGHT**

(지문): 수천 개의 서버 렉이 푸른 빛을 내뿜으며 돌아가고 있다. 한나(32)는 모니터 앞에 앉아 복잡한 데이터 스트림을 분석 중이다.

**한나**: (피곤한 듯) 서준아, 넌 대체 어디에 있는 거야...

(지문): 갑자기 화면에 'Unauthorized Access' 경고가 뜬다. 그리고 나타나는 의문의 파일 하나. 'ECHO_LOG_001'.

**S# 2. VIRTUAL SIMULATION ROOM (THE ECHO)**

(지문): 하얀 공간. 아무것도 없다. 그때, 서준(35)이 나타난다. 그는 사고 당시의 모습 그대로다.

**서준**: 한나야.

**한나**: (충격) ...서준이니? 정말 너야?

**서준**: 시간이 없어. 그들이 나를 지우려 해. 넌 지금 당장 여기서 나가야 돼.

(지문): 한나가 서준에게 손을 뻗는 순간, 서준의 형체가 디지털 노이즈로 깨지기 시작한다.

**S# 3. HANNA'S APARTMENT - NIGHT (EPISODE CLIFFHANGER)**

(지문): 한나는 집으로 돌아와 태블릿을 켠다. 서준의 목소리가 들린다.

**서준 (Digital Voice)**: 한나, 사실 말 못 한 게 있어. 사고 나기 10분 전, 내가 본 건...

(지문): 문밖에서 발소리가 들린다. 한나는 숨을 죽인다. 문고리가 천천히 돌아간다.

**BLACK OUT.**
