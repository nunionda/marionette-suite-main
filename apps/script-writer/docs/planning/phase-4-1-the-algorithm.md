# Phase 4.1: Scripting (S#1 - S#5) - THE ALGORITHM

## 1. 개요
- **목적**: 120개 씬 중 첫 5개 씬(도입부)을 집필하여 전체적인 톤과 스타일을 확립.
- **핵심 목표**: 강(KANG)의 몰락과 소피(SOPHIE)의 침투를 통한 '글리치'의 시각적 구현.

## 2. 씬별 주요 포인트
- **S#1**: 0.1%의 오차. 결벽적인 퀀트 트레이더의 세계에 균열이 가는 소리.
- **S#2**: 유령 같은 소피의 침투. 데이터가 소피를 인식하는 순간의 기괴함.
- **S#3**: 강의 자산 증발. 숫자가 사라지는 것을 지켜보는 무력함.
- **S#4**: 전 세계적 미세 정전. 알고리즘이 현실 물리 세계로 손을 뻗는 첫 신호.
- **S#5**: 강과 소피의 데이터가 동시에 말소되는 순간.

## 3. 스타일 가이드 적용
- **미장센**: 차가운 네온, 금속성 텍스처, 화면의 미세한 노이즈 연출.
- **대사**: 전문 용어를 섞은 건조하고 긴박한 티키타카(이후 접점 시).
