### Visual Language

- **Color Palette**: Deep Black `#0A0A0A` · Electric Green `#00FF41` · Blood Red `#C0392B` · Cold White `#F0F0F0`
- **Typography**: Monospace code font + Korean Gothic — code and language in collision
- **Motifs**: Streaming crypto price charts, terminal prompts, satellite-photo texture overlays
- **Mood**: Cold and urgent — human warmth visible only through the cracks

### AI Positioning (dual-layer)

- **Investor/Industry**: "World's first AI full-pipeline theatrical feature" — front and center in pitch materials, tech media briefings
- **Audience/Consumer**: Story and characters first; AI is background craft, not the product

### Pre-sale Marketing Materials Package

**One-Sheet** (1-page sales sheet)
- 2 versions: English + Korean
- Title, logline, genre, runtime, director, mood image, contact
- Print-ready for Marché + PDF

**Teaser Trailer Brief** (AI-generated footage)
- 60 seconds — Singapore exchange floor → Pyongyang bunker → code explosion sequence
- No dialogue: code + numbers + score only
- English subtitle version + Korean version
- **Production note**: Teaser production begins after final scenario revision is complete. Teaser-specific scenario to be written from final script, then shot via AI pipeline.

**Logline Sheet** (distributor mail-out)
- 3 English logline variants: long / medium / short
- Genre tags, anticipated MPAA rating, language

**Press Kit Draft**
- Project overview, director's note, AI production methodology explainer, moodboard

| 2027 Q2 | Asia OTT streaming launch |

### Festival-by-Festival Strategy

#### Cannes 2026 (May) — TOP PRIORITY

| Item | Detail |
|------|--------|
| Goal | Marché du Film pitching. Business, not competition entry. |
| Target meetings | Wild Bunch, MK2, Protagonist Pictures, Netflix EMEA, Korean OTT international divisions |
| Required materials | Pitch deck + One-Sheet + Teaser + Project Bible |
| Booth strategy | Korean Pavilion (explore KOFIC support) |
| AI angle | Separate briefing for tech journalists — "AI cinema revolution" story |

#### BIFF 2026 (October, Busan) — WORLD PREMIERE TARGET

| Item | Detail |
|------|--------|
| Goal | World premiere + domestic press buzz + Asia distributor final deals |
| Target section | New Currents or Gala Presentation |
| Strategic value | Busan premiere = Korean theatrical justification + leverage to increase OTT deal price |

#### Berlin EFM 2027 (January) — EUROPE DISTRIBUTION CLOSE

| Item | Detail |
|------|--------|
| Goal | Arrive with BIFF reception + Asia deal record. Close European independent distributors. |
| Targets | German, French, UK independent distributors |
| Positioning | "BIFF breakout, Asia-wide deals complete" — maximum social proof |

#### Toronto TIFF 2026 (September) — OPTIONAL

Deadline overlaps with BIFF. If early English-language market entry is desired, TIFF is viable. Given Asia-first strategy, BIFF is the recommendation. Choose one.
