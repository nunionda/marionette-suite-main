# Phase 3: Treatment - THE ALGORITHM

## 1. 개요
- **목적**: 8시퀀스 구조를 바탕으로 120개의 씬 로그를 설계하고, 전체 서사의 흐름을 씬 단위로 구체화.
- **핵심 목표**: `scene_logs_120.md` 작성 및 시퀀스별 트리트먼트 고도화.

## 2. 구조 설계 (120 Scenes / 8 Sequences)
각 시퀀스는 평균 **15개**의 씬으로 구성됩니다.
- **Sequence 1-2 (Act 1)**: 30 Scenes (Setup / Glitch / Alliance)
- **Sequence 3-6 (Act 2)**: 60 Scenes (Pursuit / Discovery / Descent)
- **Sequence 7-8 (Act 3)**: 30 Scenes (Crisis / Resolution)

## 3. 씬 요소 가이드라인
모든 씬 로그는 다음 요소를 포함합니다:
- **[장소/시간/등장인물/갈등/목표/주요사건/반전]**
