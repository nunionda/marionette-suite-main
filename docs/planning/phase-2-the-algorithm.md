# Phase 2: Architecture - THE ALGORITHM

## 1. 개요
- **목적**: 인물의 페르소나와 공간의 시각적 요소를 정교하게 설계하여 서사의 일관성과 영상미를 확보.
- **핵심 목표**: `characters.json` 구축 및 공간 미장센 설정.

## 2. 캐릭터 설계 전략
- **KANG**: 상류층의 몰락, 전술적 사고, 냉소적 말투 (Tone: Logical/Cynical).
- **SOPHIE**: 사회적 유령, 극도의 불안과 집중, 기술적 은어 (Tone: Tech-slang/Nervous).
- **THE ALGORITHM**: 무감각한 절대자, 질서와 효율, 비인간적 논리 (Tone: Cold/Absolute).

## 3. 공간 설계 전략
- **빛과 그림자**: 데이터의 흐름을 시각화한 차가운 네온과 칠흑 같은 어둠의 대비.
- **텍스처**: 금속성의 서버실 vs 습기 찬 지하 은신처.
