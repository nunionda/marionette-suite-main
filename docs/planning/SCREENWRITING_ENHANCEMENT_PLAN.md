## 🛠️ Step 1: Engine Logic Upgrade (Prompt Engineering)

### 1. Treatment Engine ➔ [15-Beat Roadmap]
- **변경**: 3막 8시퀀스 프롬프트 하단에 **Blake Snyder의 15비트 체크리스트**를 주입.
- **목표**: 상업 영화의 호흡(Pacing)을 110페이지 규격에 최적화.

### 2. Scenario Writer ➔ [McKee Value Checker]
- **변경**: 씬 집필 시 지문 끝에 `[SCENE VALUE: +/-]` 주석을 생성하게 하여 인물의 변화를 강제함.
- **목표**: 정적인 '설명형 씬'을 제거하고 모든 씬이 서사적 동력을 갖게 함.

### 3. Review Agent ➔ [Antagonism Audit]
- **변경**: 리뷰 에이전트에 Robert McKee의 '대항 세력의 원칙' 추가.
- **목표**: 주인공의 승리가 우연이 아닌, 강력한 적수와의 대결 끝에 얻은 값진 성취임을 검증.
